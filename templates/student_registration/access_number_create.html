How Access Numbers Work

Access numbers are automatically generated using the following format:

SLAU-YYYY-S{semester}-{reg_no_suffix}-{sequence}
Example: SLAU-2024-S1-001-00123
  • SLAU - Institution code
  • YYYY - Academic year (first part)
  • S{semester} - Semester number
  • {reg_no_suffix} - Last 3 digits of student registration number
  • {sequence} - Sequential number for the semester

The access number serves as the primary linking key for all student activities in a given academic period.
